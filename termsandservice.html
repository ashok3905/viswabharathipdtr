Appendix A: Practical tips for secure form handling

  1. Always validate inputs server-side: check that phone numbers are numeric and within expected length.
  2. Use HTTPS for all pages that host forms to prevent interception of data in transit.
  3. Use CAPTCHAs or simple bot checks if you receive spam submissions.
  4. Set a retention period and implement automated deletion scripts for old form data.
  5. Limit access: only administrators required to process enquiries should have access to submission data.
