How to Use QR Code Login

For Administrators:

• Use the pre-generated Admin QR code

• Scan with any QR code scanner app

• Direct access to admin dashboard

For Faculty Members:

• Generate individual QR codes using the "Generate Custom QR" tab

• Use format: 222p-[numbers][letter]

• Example: 222p-101a, 222p-205b

• Each faculty member gets a unique QR code

For Students:

• Generate individual QR codes for each student

• Use format: 222p-[year][section]-[roll]

• Example: 222p-1a-10 (1st year, section A, roll 10)

• Each student gets a unique QR code

Using QR Codes:

1. Open any QR scanner app on your phone

2. Point camera at the QR code

3. Tap the notification/link that appears

4. You'll be automatically logged in!

💡 Pro Tip: Print QR codes on ID cards, desk nameplates, or distribute digitally for easy access!
